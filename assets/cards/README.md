# 扑克牌图片资源

## 图片命名规则

将扑克牌图片放在此目录下，命名格式为：`{suit}-{rank}.png`

例如：
- `spades-3.png` - 黑桃3
- `hearts-14.png` - 红桃A (A的rank是14)
- `diamonds-15.png` - 方块2 (2的rank是15)
- `clubs-11.png` - 梅花J

## 花色对应
- `spades` - 黑桃 ♠
- `hearts` - 红桃 ♥
- `diamonds` - 方块 ♦
- `clubs` - 梅花 ♣

## 点数对应
- 3-10: 对应数字 3-10
- 11: J
- 12: Q
- 13: K
- 14: A
- 15: 2

## 图片尺寸建议
- 推荐尺寸：120x168 像素
- 格式：PNG（支持透明背景）

## 背面图片
- 文件名：`back.png` - 用于显示AI手牌的背面
