# AI中控系统快速启动指南

## 🚀 快速开始

### 1. 启动应用

```bash
npm run dev
```

### 2. 打开浏览器

访问显示的地址（通常是 `http://localhost:5173`）

### 3. 打开AI中控系统

1. 在浏览器中找到右下角的 **"🧠 AI中控"** 按钮
2. 点击打开控制面板

## 🔧 常见问题

### 问题1: 按钮不显示
**解决方案**:
- 检查是否在App.tsx中导入了AIControlDashboard
- 检查浏览器控制台是否有错误
- 刷新页面

### 问题2: 数据不显示
**解决方案**:
- 检查AI中控系统是否已初始化
- 查看浏览器控制台错误信息
- 确认SystemApplication已启动

### 问题3: 样式不正确
**解决方案**:
- 检查CSS文件是否正确加载
- 清除浏览器缓存
- 检查是否有样式冲突

### 问题4: 功能不工作
**解决方案**:
- 查看浏览器控制台错误
- 检查相关服务是否正确实现
- 确认事件订阅是否正确

## 📊 系统状态检查

### 检查初始化状态

打开浏览器控制台，运行：

```javascript
// 检查AI中控系统
const aiControl = window.__AI_CONTROL__; // 如果暴露了全局变量
console.log('AI中控系统:', aiControl);

// 或者通过React DevTools检查组件状态
```

### 检查监控状态

在AI中控面板中：
1. 打开"仪表盘"标签
2. 查看"系统状态"卡片
3. 确认显示"✅ 运行中"或"⏸️ 已停止"
