## 🔍 问题排查

### 如果按钮不显示

1. **检查App.tsx**
   ```typescript
   import { AIControlDashboard } from './components/ai-control/AIControlDashboard';
   // ...
   <AIControlDashboard />
   ```

2. **检查CSS加载**
   - 确认CSS文件存在
   - 检查CSS导入路径

3. **检查控制台错误**
   - 打开浏览器开发者工具
   - 查看Console标签
   - 查找错误信息

### 如果控制面板不打开

1. **检查组件导入**
   - 确认所有子组件已导入
   - 检查导入路径

2. **检查状态管理**
   - 确认useState正常工作
   - 检查事件处理

3. **检查样式**
   - 确认CSS文件已加载
   - 检查z-index设置

### 如果数据不显示

1. **检查AI中控系统初始化**
   - 查看控制台日志
   - 确认SystemApplication已启动
   - 确认AIControlModule已初始化

2. **检查服务可用性**
   - 确认InteractionService可用
   - 确认AIControlCenter已初始化

3. **检查数据源**
   - 确认监控已启动
   - 确认有数据被收集
