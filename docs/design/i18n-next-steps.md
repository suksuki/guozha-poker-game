### 选项 2: 渐进式迁移

**逐步将现有组件迁移到新框架**

**迁移优先级：**

1. **简单组件**（先迁移）
   - `ErrorScreen` - 少量翻译
   - `LanguageSwitcher` - 使用语言 Hook
   - 其他小型组件

2. **中等组件**（其次）
   - `GameConfigPanel` - 使用多个命名空间
   - `GameResultScreen` - 游戏结果展示
   - 其他中等规模组件

3. **复杂组件**（最后）
   - `MultiPlayerGameBoard` - 大型组件
   - `CardTrackerPanel` - 复杂功能
   - 其他大型组件

**迁移步骤：**
1. 选择一个组件
2. 运行 `npm run i18n:scan --component ComponentName`
3. 生成翻译文件（如需要）：`npm run i18n:generate --component ComponentName`
4. 更新组件代码使用新的 Hooks
5. 测试功能正常
6. 重复步骤 1-5
