# 方位布局设计：东南西北方向

## 🎯 核心思路

利用**4人游戏 = 东南西北四个方向**，让布局更直观、更符合传统牌桌布局！

---

## 🎮 4人布局设计

### 布局方案

#### 方案1：传统牌桌布局 ⭐⭐⭐⭐⭐（推荐）

```
         北（AI对手1）
            👤
    
    👤                   👤
 西（AI队友）       东（AI对手2）
    

        南（你）👤
```

**优点**：
- ✅ 符合传统牌桌习惯
- ✅ 直观易懂
- ✅ 方向感强

**座位对应**：
- 南：玩家（你）- 底部，方便操作
- 北：对手1 - 顶部
- 东：对手2 - 右侧
- 西：队友 - 左侧（和你在同一边）

### UI设计要点

#### 1. 视觉层次

```
【你的位置 - 南】
- 底部中央，最大最显眼
- 手牌区域最大
- 操作按钮最明显

【队友 - 西】
- 左侧，中等大小
- 有明显的"队友标识"（如蓝色边框）
- 显示团队分数

【对手 - 北、东】
- 顶部和右侧
- 相对较小（因为不需要操作）
- 显示对方团队分数
```

#### 2. 团队视觉区分

```
团队A（你和队友）：
- 蓝色主题
- 左侧和底部
- 共同的团队分数显示

团队B（对手1和对手2）：
- 红色主题
- 顶部和右侧
- 共同的团队分数显示
```

#### 3. 信息展示

```
每个位置显示：
- 玩家头像/名称
- 手牌数量
- 当前状态（出牌中/等待/要不起）
- 团队标识

中心区域：
- 出牌区域（所有玩家出的牌都显示在这里）
- 当前轮次分数
- 团队分数对比
```

## 🎮 6人布局设计

### 布局方案

#### 方案1：六边形布局 ⭐⭐⭐⭐⭐（推荐）

```
         👤
     北（对手1）
    
👤                👤
西（队友1）    东（对手2）
   
   👤
 西北（对手3）

         👤
      南（你）
     
    👤
  东南（队友2）
```

**座位对应**：
- 南：玩家（你）- 底部中央
- 北：对手1 - 顶部
- 东：对手2 - 右侧
- 西：队友1 - 左侧
- 西北：对手3 - 左上
- 东南：队友2 - 右下

#### 方案2：对称布局 ⭐⭐⭐⭐

```
     👤       👤
  对手1    对手2
     
👤              👤
队友1        对手3

   👤       👤
  你      队友2
```

**座位对应**：
- 底部左侧：玩家（你）
- 底部右侧：队友2
- 中间左侧：队友1
- 顶部左侧：对手1
- 顶部右侧：对手2
- 中间右侧：对手3

### 团队视觉区分（6人）

```
团队A（你 + 2个队友）：
- 蓝色主题
- 左下、左中、右下
- 共同的团队分数

团队B（3个对手）：
- 红色主题
- 左上、右上、右中
- 共同的团队分数
```

### 1. 信息共享 + 方向布局

```

- 从西方向滑入信息气泡
```

## 🎯 具体实现建议

### 1. 布局组件结构

```
<GameBoard>
  <PlayerPosition direction="north" team="teamB" />
  <PlayerPosition direction="west" team="teamA" />
  <PlayerPosition direction="east" team="teamB" />
  <PlayerPosition direction="south" team="teamA" isMainPlayer />
  
  <CenterArea>
    <PlayArea />
    <ScoreDisplay />
  </CenterArea>
</GameBoard>
```

### 2. CSS布局

```css
.game-board {
  display: grid;
  grid-template-areas:
    ". north ."
    "west center east"
    ". south .";
  grid-template-columns: 1fr 2fr 1fr;
  grid-template-rows: 1fr 2fr 1fr;
}

.player-north { grid-area: north; }
.player-west { grid-area: west; }
.player-east { grid-area: east; }
.player-south { grid-area: south; }
.center-area { grid-area: center; }
```

### 3. 动画配置

```typescript
interface DirectionAnimation {
  from: 'north' | 'south' | 'east' | 'west';
  to: 'center' | 'north' | 'south' | 'east' | 'west';
  animation: AnimationType;
}

// 出牌动画
playCardAnimation: {
  from: playerDirection,
  to: 'center',
  type: 'fly'
}

// 信息动画
messageAnimation: {
  from: senderDirection,
  to: receiverDirection,
  type: 'slide'
}
```

## 📊 4人 vs 6人对比

### 4人模式优势

- ✅ 布局最清晰（东南西北）
- ✅ 方向感最强
- ✅ 团队2v2，平衡最好
- ✅ 实现最简单

### 6人模式优势

- ✅ 更热闹（3v3）
- ✅ 策略更复杂
- ✅ 需要更多配合

### 推荐

**先实现4人模式**，因为：
- 布局最直观
- 方向标识最清晰
- 实现相对简单
- 用户体验最好

**后续扩展6人模式**，使用六边形布局。
