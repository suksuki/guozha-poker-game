# 4人模式方向性出牌布局细节

## 已确认方案

- ✅ **出牌位置**：方案B - 出牌在玩家头像旁边（固定偏移）
- ✅ **多次出牌**：选项B - 按时间顺序依次排列
- ✅ **中心区域**：保留 PlayArea 显示最后出牌

## 具体布局设计

### 1. 北边玩家（顶部）

```
      [方向标签：北]
      [玩家头像]
      ↓ 偏移距离（如 30px）
[出牌1] [出牌2] [出牌3] (横向排列，从左到右)
```

**布局细节：**
- 头像在顶部
- 出牌区域在头像**下方**
- 横向排列（flexDirection: 'row'）
- 多次出牌从左到右依次排列

### 2. 东边玩家（右侧）

```
[出牌1] → [玩家头像] [方向标签：东]
[出牌2] →       (右侧)
[出牌3] →
```

**布局细节：**
- 头像在右侧
- 出牌区域在头像**左侧**
- 纵向排列（flexDirection: 'column'）
- 多次出牌从上到下依次排列

### 3. 西边玩家（左侧）

```
       ← [方向标签：西] [玩家头像]
       ← [出牌1]
       ← [出牌2]
       ← [出牌3]
```

**布局细节：**
- 头像在左侧
- 出牌区域在头像**右侧**
- 纵向排列（flexDirection: 'column'）
- 多次出牌从上到下依次排列

### 4. 南边玩家（底部 - 人类玩家）

```
[出牌1] [出牌2] [出牌3] (横向排列，在按钮组上方)
        [按钮组]
        [手牌区域]
```

**布局细节：**
- 头像不显示（已在手牌左侧显示）
- 出牌区域在**按钮组上方**（固定位置）
- 横向排列（flexDirection: 'row'）
- 多次出牌从左到右依次排列

## 偏移距离建议

- **北边/南边**：头像到出牌区域的垂直距离，建议 30-50px
- **东边/西边**：头像到出牌区域的水平距离，建议 30-50px

## 排列间距

- **横向排列**（北、南）：每次出牌之间间距 15-20px
- **纵向排列**（东、西）：每次出牌之间间距 15-20px

## 卡牌大小

建议统一使用小尺寸卡片（如 `size="small"`，约 40px 宽），与当前 RoundPlaysPanel 保持一致。

## 实现细节

每个玩家位置需要：
1. 玩家头像区域（已有）
2. 出牌容器区域（新增）
   - 根据方向设置 flexDirection
   - 根据方向设置偏移位置
   - 显示该玩家的所有出牌记录（按时间顺序）

## 需要确认

1. **南边人类玩家出牌位置**（选项A/B/C）
2. **偏移距离具体数值**（建议30-50px）
3. **排列间距具体数值**（建议15-20px）
4. **卡牌大小**（建议small，40px宽）
