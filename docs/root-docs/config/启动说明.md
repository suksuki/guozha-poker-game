# 服务启动说明

## 快速启动（推荐）

```bash
cd /Users/jin/dev/dev/guozha
./start-all-services.sh
```

## 分别启动

### 1. 启动 Piper TTS 服务（后台）

```bash
cd /Users/jin/dev/dev/guozha
source venv-piper/bin/activate
nohup python3 scripts/piper-tts-server.py > /tmp/piper-tts.log 2>&1 &
```

### 2. 启动前端服务（前台）

```bash
cd /Users/jin/dev/dev/guozha
npm run dev
```

## 检查服务状态

```bash
# 检查端口占用
lsof -ti:5000 && echo "✅ Piper TTS运行中" || echo "❌ Piper TTS未运行"
lsof -ti:3000 && echo "✅ 前端服务运行中" || echo "❌ 前端服务未运行"

# 检查Piper TTS健康状态
curl http://localhost:5000/health
```

## 停止服务

```bash
# 停止Piper TTS
pkill -f "piper-tts-server.py"

# 停止前端服务
pkill -f "vite"
```

## 查看日志

```bash
# Piper TTS日志
tail -f /tmp/piper-tts.log
```

## 访问地址

- 前端应用：http://localhost:3000
- Piper TTS服务：http://localhost:5000
- Piper TTS健康检查：http://localhost:5000/health
