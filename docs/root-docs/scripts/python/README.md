# Python 脚本

本目录包含Python脚本文件。

## 脚本说明

- `start-melo-tts-server.py` - 启动MeLo TTS服务器
- `move_docs.py` - 文档移动工具脚本

## 使用方式

```bash
# 启动MeLo TTS服务器
python docs/root-docs/scripts/python/start-melo-tts-server.py

# 运行文档移动工具
python docs/root-docs/scripts/python/move_docs.py
```

## 注意事项

- 确保已安装所需的Python依赖
- 某些脚本可能需要虚拟环境
- 检查脚本中的路径配置是否正确
