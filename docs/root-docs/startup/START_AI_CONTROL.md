# 🚀 AI中控系统启动指南

## 快速启动

### 1. 启动开发服务器

```bash
npm run dev
```

### 2. 打开浏览器

访问显示的地址（通常是 `http://localhost:5173`）

### 3. 打开AI中控系统

1. 在页面右下角找到 **"🧠 AI中控"** 按钮
2. 点击打开控制面板
3. 开始测试各项功能
