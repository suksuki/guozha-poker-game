# 测试文件目录

本目录包含各种测试相关的文件。

## 目录结构

```
test-files/
├── audio/          # 音频测试文件（.wav）
├── json/           # JSON测试文件和测试结果
└── logs/           # 测试日志
```

## 文件说明

### audio/
包含音频测试文件，用于TTS和语音功能测试。

### json/
包含测试用的JSON配置文件和测试结果文件：
- `test-*.json` - 测试配置文件
- `test-results.txt` - 测试结果
- `merged-tests-*.test.ts` - 合并的测试文件

### logs/
包含测试运行日志。

## 注意事项

- 这些文件主要用于开发和测试
- 可以定期清理旧文件以节省空间
- 重要测试结果应归档到 `docs/tests/` 目录
